The programs mergesort.py and quicksort.py first asks the size of array, after which user enters array item.Enter one integer item and hit ENTER to insert the next item of array
The programs mergesortTime.py and quicksortTime.py are programs that measures time complexity for sorted array sizes 1000,1500 and 2000 or sorted array from size 0 to 2000 in steps of 100
#Environment dependency :
python 3.6.2  [MSC v.1900 64 bit (AMD64)] on win32
